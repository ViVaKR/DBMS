# MSSQL

## Installing mssql-cli

```bash
    # install pip
    sudo easy_install pip

    # Update pip
    python -m pip install --upgrade pip

    # Install mssql-cli
    sudo pip install mssql-cli

    # Run mssql-cli
    mssql-cli


    # Uninstalling mssql-cli
    sudo pip uninstall mssql-cli
```
